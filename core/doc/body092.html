List getDataListFromPath(String path, Object userData);

基本的なメソッドの一つです。
指定のPath上にあるデータをリスト形式で返します。
一覧表示するときなどに利用します。

このメソッドの処理速度を上げるときの方法は
・整数型の主キー
・selectManyを設定する
これで処理速度は格段に向上します。

List getDataListFromPathAndUser(String path, Object userData);

上記のメソッドとほぼ同じですが、userDataで指定されたユーザが最後に操作(Invoke)したデータが表示されます。
