List getDataIDFromPath(String path, Object userData);
データのIDのみをリスト形式で返します。
単なる一覧ではなく、検索条件を指定してその値だけを表示したい時などに使います。

このメソッドが使える条件は
データの主キーが整数型
です。
その他の場合に使ったときの動作は不定です。

List getDataIDFromPathAndUser(String path, Object userData);

上記のメソッドとほぼ同じですが、userDataで指定されたユーザが最後に操作(Invoke)したデータが表示されます。
